# DarkPDF
Convert your PDF files to dark mode! Ease your eye stress at night.

</p>


## Requirements
* Numpy
* PIL
* pdf2image

## Usage

1. Drag DarkPDF.py with your pdf file into a directory.

2. Run
```
python3 DarkPDF.py your.pdf
```
